DROP-IN SLOTS — Minecraft font

minecraft.woff2
  The Minecraft display font for the brand title and headings.
  Recommended source: Minecraftia (dafont.com, free) or Mojangles.

Convert TTF/OTF to WOFF2 first (Google "ttf to woff2 converter"
or use fonttools locally). WOFF2 is ~half the file size.

If this file is missing, the page falls back to a clean pixel
fallback chain (Courier New, monospace), and the brand still
reads correctly. No code change needed when you add it.

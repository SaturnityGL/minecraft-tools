BlockForge FAQ

Common questions about BlockForge. If yours is not here, the guide probably covers it.

What is BlockForge?

BlockForge is a free browser tool that converts any image into Minecraft block art. It maps each pixel of your image to the closest matching block from 263 supported textures and exports a .schematic file you can import into Litematica or WorldEdit.

How do I import the schematic into Minecraft?

You need either Litematica (recommended for survival) or WorldEdit (creative). Install the mod for your Minecraft version, drop the .schematic file into the mod's schematic folder, open it from inside the mod, and place it where you want to build. The full guide walks through both options.

What Minecraft versions are supported?

The .schematic format BlockForge exports works with every modern version of Minecraft Java Edition that supports WorldEdit or Litematica, which is essentially everything from 1.7 onward. Bedrock Edition does not support .schematic files natively, you would need a conversion tool.

Is BlockForge free?

Yes. There is no paywall, no signup, no email gate, no watermark. BlockForge is funded by Ko-Fi donations and ads on the main Saturnity site. The app itself stays free.

Can I use BlockForge in survival mode?

Yes, through Litematica. Litematica shows a transparent preview of the schematic in your world, lists every block you need, and lets you place them one at a time. It does not auto-build in survival, that would be cheating. It just makes the manual build a lot more accurate.

How many blocks does BlockForge support?

263 unique blocks across wool, concrete, concrete powder, terracotta, glazed terracotta, wood, stone, ore, nature, nether, end, glass, and miscellaneous decorative blocks. The full list with textures and IDs is on the blocks reference page.

What image formats does BlockForge accept?

PNG, JPG, GIF, WEBP, and BMP. Transparent PNGs work best for logos and character art, since the transparent background does not get matched to a block. For photos and landscapes, JPG is fine.

Does BlockForge work on mobile?

The app runs in mobile browsers, but the canvas interface is built for mouse precision. Phones can technically generate a schematic, but you cannot import that schematic into Bedrock Minecraft and Java Edition does not run on phones, so the result is not directly useful from a phone-only workflow. Use a laptop or desktop.
